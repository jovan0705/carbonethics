# FRONTEND
## Installation
```
1. cd carbonethics-fe/
2. nvm use 19 - Make sure you have used node version > 18
3. npm i (to install dependencies)
4. npm run dev (to run the app)
```
# BACKEND
## Installation
```
1. cd server/
2. Make sure you have python installed
3. pip install django (to install django)
4. pip install djangorestframework (to install django rest framework)
5. Change the database configuration on /carbonethics/settings.py DATABASES to your local settings
6. python manage.py migrate (to migrate the tables to database)
7. python manage.py runserver (to run the server)
```
